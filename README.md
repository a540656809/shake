# shake
h5摇一摇小游戏  
描述： 摇一摇排行榜  
采用技术 socket.io html redis spring boot

# 使用方法
1.运行ShakeApplication.java方法后,访问localhost/shake.html	

2.输入admin为控制操作台

3.输入其他名称为玩家

4.admin管理员点击开始后,玩家即可开始摇晃手机来增加分数。摇晃途中，排行实时变化，当管理员点击结束时，本次摇一摇比赛结算分数排行。
